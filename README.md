# smita--milestone1
🚀 PrognosAI Project – NASA CMAPSS RUL Prediction

This project predicts the Remaining Useful Life (RUL) of aircraft engines using the NASA CMAPSS dataset.
It helps in predictive maintenance – knowing in advance when an engine might fail.

📂 Project Folders

data/ → Raw dataset files

converted_csv/ → Cleaned & converted CSV files

models/ → Saved trained models

scripts/ → Python scripts for each step

README.md → Project details

requirements.txt → Needed Python libraries

⚙️ Steps in the Project

Convert NASA dataset (TXT → CSV)

Do feature engineering (create useful features)

Train Machine Learning models

Evaluate results (MSE, RMSE, Accuracy)

Save the best model

🛠 Tools & Libraries

Python

Pandas, NumPy (data processing)

Scikit-learn (machine learning)

Matplotlib (plots & graphs)

📊 Output

CSV files ready for training

Predicted Remaining Useful Life (RUL)

Performance report of models

Stored trained models
